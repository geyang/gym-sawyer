# `gym-Sawyer`, a Sawyer plugin for OpenAI Gym

Current release at `v0.1.0`.

## Usage Example

```python
import gym

env = gym.make("gym_sawyer:PickPlace-v0", width=84, height=84, cam_id=-1)

env.render("notebook")
```
